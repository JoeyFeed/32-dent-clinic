# Исправление ошибки Vite/esbuild

## Проблема
Ошибка: `Error: The service was stopped` при запуске `npm run dev` в frontend.

Это часто происходит из-за:
- Путей с пробелами и кириллицей
- Проблем с esbuild на Windows
- Антивируса, блокирующего esbuild

## Решения

### Решение 1: Переустановить зависимости (попробуйте сначала)

```powershell
cd frontend
rm -r node_modules
rm package-lock.json
npm install
npm run dev
```

### Решение 2: Обновить Vite и зависимости

```powershell
cd frontend
npm install vite@latest @vitejs/plugin-react@latest --save-dev
npm run dev
```

### Решение 3: Добавить исключение в антивирус

1. Откройте настройки антивируса/Защитника Windows
2. Добавьте исключение для папки проекта
3. Или добавьте исключение для `node_modules\esbuild`

### Решение 4: Использовать альтернативный порт

Если порт 5173 занят, измените в `vite.config.js`:
```js
server: {
  port: 3001, // или другой свободный порт
}
```

### Решение 5: Запустить от имени администратора

1. Закройте текущий терминал
2. Откройте PowerShell от имени администратора
3. Перейдите в папку frontend
4. Запустите `npm run dev`

### Решение 6: Использовать другой порт и настройки

Если ничего не помогает, попробуйте запустить с флагами:

```powershell
cd frontend
npx vite --host --port 5173
```

---

## Быстрая проверка

Проверьте, что все зависимости установлены:

```powershell
cd frontend
npm list vite @vitejs/plugin-react
```

Должны быть установлены версии.

---

## Если ничего не помогает

Попробуйте создать проект заново в папке без пробелов:

1. Создайте папку `C:\Projects\32-dent-clinic`
2. Скопируйте туда проект
3. Запустите из новой папки
